## 🧬 About This Repository

<img align="right" width="300" src="https://media.giphy.com/media/qgQUggAC3Pfv687qPC/giphy.gif"/>

This repository is a **curated collection** of my solutions to Data Structures & Algorithms problems sourced from the world's top competitive programming platforms.

Every solution reflects my approach to **breaking down complex problems** into clean, efficient, and well-commented code.

Problems are organized by **difficulty level** — making it easy to navigate, revisit, and **track growth over time**.

## 🚀 Getting Started

```bash
# 📥 Clone the repository
git clone https://github.com/arryansharma1/dsa-practice.git

# 📂 Navigate into the folder
cd dsa-practice

# 🟢 Browse by difficulty
cd Easy/arrays

# 🏃 Run a solution (C++ example)
g++ solution.cpp -o solution && ./solution
```
